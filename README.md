# L7mo ISchool Project

مرحباً بك في مشروع iSchool! 

بما أن هناك بعض الملفات الضخمة والحساسة التي لا يتم رفعها على GitHub (بسبب حجمها أو سريتها)، يرجى اتباع هذه الخطوات لتشغيل المشروع بنجاح على جهازك:

## 1️⃣ إعداد ملفات البيئة (Environment Variables)
المشروع يعتمد على متغيرات بيئة لكل من الـ Backend والـ Frontend:

- **في مسار `backend/`:** 
  قم بنسخ ملف `.env.example` وأعد تسميته إلى `.env`، ثم ضع به مفاتيح الـ API الخاصة بك (GROQ و PEXELS).
  
- **في مسار `frontend/`:**
  قم بنسخ ملف `.env.example` وأعد تسميته إلى `.env.local`، ثم ضع به إعدادات الـ Firebase الخاصة بالمشروع.

- **ملف خدمة فايربيس:**
  اطلب ملف `firebase-key.json` من مدير المشروع وضعه بداخل مجلد `backend/`.

## 2️⃣ إعداد موديلات الذكاء الاصطناعي (AI Models)
ملفات الذكاء الاصطناعي (مثل SadTalker و EdgeTTS و GFPGAN) حجمها يتخطى الـ 2 جيجابايت ولذلك غير مرفوعة هنا.
يجب عليك الحصول على مجلد `backend/models/` بالإضافة إلى `backend/SadTalker/checkpoints/` و `backend/gfpgan/weights/` من مدير المشروع ووضعها في نفس مساراتها بالضبط.

## 3️⃣ تشغيل المشروع باستخدام Docker
تأكد من تثبيت وتشغيل برنامج **Docker Desktop**، ثم افتح الـ Terminal في نفس مسار هذا الملف واكتب:

```bash
docker-compose up --build
```

سيعمل الـ Backend على بورت `8000`، والـ Frontend على بورت `3000`.
